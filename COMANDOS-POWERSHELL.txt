COMO APLICAR O PATCH DE PRODUTOS

1) Feche o servidor, se estiver rodando:
CTRL + C

2) Entre na pasta do projeto:
D:
cd \Projetos\gestor-restaurante

3) Faça backup do arquivo atual:
if (Test-Path .\src\app\produtos\page.tsx) { Copy-Item .\src\app\produtos\page.tsx .\src\app\produtos\page.backup.tsx -Force }

4) Descompacte o ZIP baixado dentro da raiz do projeto:
Expand-Archive -Path "$env:USERPROFILE\Downloads\patch-produtos-botao-voltar.zip" -DestinationPath "D:\Projetos\gestor-restaurante" -Force

5) Limpe o cache do Next:
Remove-Item -Recurse -Force .next

6) Teste o build:
npm run build

7) Se passar, teste local:
npm run dev

8) Abra no navegador:
http://localhost:3000/produtos
http://localhost:3000/configuracoes

9) Se tudo estiver certo, publique:
git add .
git commit -m "adiciona botao voltar em produtos"
git push
